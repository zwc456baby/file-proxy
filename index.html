使用本站服务器加速前缀: https://pd.zwc365.com/seturl/

Cloudflare Workers 加速前缀: https://pd.zwc365.com/cfworker/

支持命令行，可在 Git Bash 中 clone，或终端 wget 代下(详情见项目说明)

CloudFlare 代下无速度及文件大小限制，服务器代下限制文件大小 2G

本站现在有 Chrome 扩展 ，安装扩展后，使用鼠标右键即可选择线路加速下载文件

服务限制十个线程及文件分片，若使用 IDM 等多线程下载工具，请降低线程数

欢迎小伙伴们 收藏和分享 本服务

如果是网页使用，直接将原链接粘贴到输入框点击下载即可加速。

如果使用其他工具进行文件下载，则在原链接前新增加速前缀：

Git 项目加速：git clone https://pd.zwc365.com/seturl/< https 开头的项目地址>

Wget 加速下载：wget https://pd.zwc365.com/seturl/<原来的文件链接>

google 或 jcenter 仓库镜像：https://pd.zwc365.com/cfworker/<仓库地址>

加速任意链接：https://pd.zwc365.com/seturl/<任意 http 或 https 链接>

在后面拼接的URL必须是 http:// 或者 https:// 开头

合格的URL 1: https://github.com/zwc456baby/file-proxy.git

合格的URL 2: https://github.com/zwc456baby/file-proxy/archive/master.zip

不合格的示例: github.com/zwc456baby/file-proxy/archive/master.zip

不合格的示例: www.baidu.com
